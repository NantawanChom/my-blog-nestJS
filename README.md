# My-Blog-nestJS ✏️
Simple blog create by nestJS and mongoDB

---

## Quick run

```bash
cp env-example .env
npm install

npm run start
npm run start:prod

npm run start:dev
npm run start:debug

brew services start mongodb-community@6.0
brew services stop mongodb-community@6.0

mongosh

```

## Tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e
```
